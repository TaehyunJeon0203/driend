# Driend 개발 계획

> Drive + Friend. 내 드라이브를 기록하고 친구와 함께 달리는 앱.

---

## 앱 컨셉

한국 타겟 드라이빙 앱. 주행 경로를 지도에 히트맵으로 기록하고, 시/군 단위 도장깨기와 친구 랭킹 기능을 제공한다.

---

## 기술 스택

| 분야 | 선택 |
|------|------|
| 프레임워크 | Expo (React Native) + TypeScript |
| 라우팅 | Expo Router (파일 기반) |
| 지도 | Mapbox (@rnmapbox/maps) |
| 백엔드/DB | Supabase (PostgreSQL + PostGIS) |
| 인증 | 익명 로그인 → 카카오 로그인 (비즈니스 인증 후) |
| 상태관리 | Zustand |
| 플랫폼 | iOS + Android |

## 화면별 UX 방향

### 1. 지도 화면 (메인)

**히트맵 레이어 (C안)**
- 줌 아웃 (한국 전체): B 지역 열감형 — 자주 간 지역이 얼룩처럼 퍼지는 히트맵
- 줌 인 (도시 레벨): A 경로 강조형 — 실제 달린 도로가 선으로 빛남 (밀도에 따라 파랑→빨강)
- 두 레이어 동시 존재, 줌 레벨로 자동 전환

**도장깨기 레이어**
- 한국 시/군 GeoJSON 경계를 지도 위에 오버레이
- 미방문: 기본 색 (반투명)
- 방문 (= 사진 등록): 등록한 사진이 경계 모양으로 클리핑되어 채워짐
- 사진 없이 방문만 한 경우: 포인트 컬러로 채워짐
- **방문 기준: 사진 수동 등록 시점** (GPS 자동 감지 없음)

**GPS 추적 방향**
- ❌ 수동 "주행 시작" 버튼 방식 (현재 구현, 임시)
- ✅ 목표 1: 다른 내비 앱 (카카오맵, 티맵 등)에서 경로 데이터 가져오기
- ✅ 목표 2: 자동 주행 감지 (속도/가속도 기반, 백그라운드)
- → 두 방식 중 기술적으로 가능한 걸 우선 구현

---

### 2. 통계 화면

중요도 순:
1. **누적 총 거리** (km) — 상단 히어로 숫자
2. **월별 거리 그래프** — 막대 그래프, 최근 6~12개월
3. **방문 도시 수** — 사진 등록한 시/군 개수
4. **최근 주행 목록** — 날짜 / 거리 / 소요시간
5. **등록 차량** — 차량 이름 + 사진 썸네일

---

### 3. 프로필 / 차량

**프로필**
- 닉네임, 프로필 사진
- 카카오 계정 연동 상태 표시

**차량 등록 (A안 - 간단)**
- 차종 이름 (텍스트) + 차량 사진
- 현재 1대만 지원, 추후 다대 확장 가능하도록 DB는 vehicles 테이블로 관리

---

### 4. 랭킹 화면

**범위**: 친구 탭 / 전체 탭 전환

**카테고리**
- 누적 총 거리
- 방문 도시 수 (사진 등록 기준)
- 최고 속도
- 최장 단일 주행 거리
- 야간 주행 횟수 (22시~06시)
- 이달의 거리

---

### 5. 친구 기능

친구 추가 방법 3가지:
- **A. 닉네임 검색** — 앱 내 검색
- **B. 카카오 친구** — 카카오 연동 후 카카오 친구 자동 불러오기
- **C. 초대 링크** — 링크 공유로 친구 추가

친구 신청 → 수락 방식
